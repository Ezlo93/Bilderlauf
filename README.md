# Bilderlauf
Loads a .bmp file and creates a landscape of hexagons, on which the user can move. This is done using a really old OpenGL version because it was a requirement.

W,A,S,D - move

Space - jump

Mouse - look around in first person view

1 - toggle wireframe

2 - toggle anaglyph
